# 🎯 ДЕМО: Доска объявлений КликТорг

## 🚀 Быстрый старт

### 1. Клонирование и установка
```bash
git clone <repository-url>
cd board-app
npm run install:all
```

### 2. Запуск приложения
```bash
npm run dev
```

Открыть в браузере: **http://localhost:5173**

## 🎨 Интерфейс и функциональность

### 🏠 Главная страница
- **Современный дизайн** с градиентами и анимациями
- **Фильтрация объявлений** по категории, цене, местоположению
- **Поиск по тексту** в названии и описании
- **Сортировка** (новые, старые, по цене)
- **Пагинация** для удобного просмотра
- **Адаптивный дизайн** для мобильных устройств

### 🔐 Система аутентификации

#### Регистрация (/register)
- Валидация данных
- Проверка уникальности email/username
- Хеширование паролей
- Автоматический вход после регистрации

#### Вход (/login)
- Безопасная авторизация
- JWT токены
- Запоминание состояния

### 📝 Создание объявлений (/create-ad)
- **Форма с валидацией** всех полей
- **Загрузка до 10 изображений** (до 5MB каждое)
- **Превью изображений** с возможностью удаления
- **Автозаполнение категорий** из базы данных
- **Счетчик символов** для описания

### 👀 Детальный просмотр (/ad/:id)
- **Галерея изображений** с навигацией
- **Миниатюры** для быстрого переключения
- **Полная информация** об объявлении
- **Контакты продавца**
- **Статус объявления**

### 👤 Профиль пользователя
- **Меню пользователя** в хедере
- **Аватар** с инициалами
- **Быстрый доступ** к функциям

## 🛠 Технические особенности

### Backend (Node.js + Express)
- ✅ RESTful API
- ✅ JWT аутентификация
- ✅ Валидация данных (Mongoose)
- ✅ Загрузка файлов (Multer)
- ✅ Безопасность паролей (bcrypt)
- ✅ CORS настроен
- ✅ Error handling

### Frontend (React + TypeScript)
- ✅ Современный UI/UX дизайн
- ✅ TypeScript типизация
- ✅ React Router навигация
- ✅ Context API для состояния
- ✅ Axios для HTTP запросов
- ✅ SCSS модули
- ✅ Responsive дизайн

### База данных (MongoDB)
- ✅ Схемы с валидацией
- ✅ Связи между коллекциями
- ✅ Индексы для оптимизации
- ✅ Middleware для очистки

## 🎯 Основные сценарии использования

### 1. Просмотр объявлений (гость)
1. Заходим на главную страницу
2. Видим список объявлений с фильтрами
3. Применяем фильтры по категории/цене
4. Кликаем на объявление для детального просмотра
5. Видим контакты продавца

### 2. Регистрация и создание объявления
1. Нажимаем "Регистрация"
2. Заполняем форму регистрации
3. Автоматически попадаем на главную (авторизованы)
4. Нажимаем "Разместить объявление"
5. Заполняем форму с загрузкой фото
6. Публикуем объявление

### 3. Управление профилем
1. Кликаем на аватар в хедере
2. Выбираем пункт меню
3. Редактируем профиль
4. Просматриваем свои объявления
